# Rainmeter To Do List Skin

> Made just for fun!  
> I wanted a simple to-do list always visible on my workspace, just like on Linux.  
> After discovering Rainmeter, I decided to create my own to-do list skin, even though I knew nothing about Lua.  
> With the help of AI, it became a one-day challenge and a great learning experience.  
> Now I know the basics of Lua and almost nothing of how Rainmeter scripting works!

## Features

- Minimalistic and dynamic to-do list for Windows desktop
- Customizable font, size, colors, and button styles
- Daily, important, and recurring task support
- Trash/undo for deleted tasks
- Settings panel for easy configuration

## How to Use

1. Install [Rainmeter](https://www.rainmeter.net/)
2. Copy this skin folder to your Rainmeter `Skins` directory
3. Load the skin via Rainmeter
4. Enjoy your custom to-do list!
